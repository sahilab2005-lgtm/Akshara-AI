# Akshara AI

A multilingual AI assistant built using FastAPI, Sarvam LLM and Tesseract OCR.

## Features

- Text Generation
- Text Summarization
- Text Translation
- OCR Text Extraction from Images
- File Upload Support
- FastAPI Backend
- HTML, CSS, JavaScript Frontend

## Tech Stack

- Python
- FastAPI
- Transformers
- PyTorch
- Tesseract OCR
- HTML
- CSS
- JavaScript

## Project Structure

```
Akshara-AI/
│
├── main.py
├── requirements.txt
├── README.md
│
├── templates/
│   └── chat.html
│
├── static/
│   ├── style.css
│   └── script.js
│
├── uploads/
│
└── sarvam_local/
```

## Installation

```bash
pip install -r requirements.txt
```

## Run

```bash
uvicorn main:app --reload
```

Open:

```
http://127.0.0.1:8000/chat
```
