# Akshara AI

A multilingual AI assistant built using FastAPI, Sarvam LLM and Tesseract OCR.

## Features

- Text Generation
- Text Summarization
- Text Translation
- OCR Text Extraction from Images
- File Upload Support
- FastAPI Backend
- HTML, CSS, JavaScript Frontend

## Tech Stack

- Python
- FastAPI
- Transformers
- PyTorch
- Tesseract OCR
- HTML
- CSS
- JavaScript
